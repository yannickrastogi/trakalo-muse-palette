# Music Industry Knowledge Skill

## Workflow standard d'une track
1. Création en studio (beat, topline, recording)
2. Mix & mastering
3. Metadata (splits, credits, ISRC, UPC)
4. Registration aux sociétés de gestion (PROs + droits voisins)
5. Pitching (labels, sync supervisors, playlists)
6. Distribution (si release)
7. Collecte royalties (3 flux distincts)

## Les 3 types de droits musicaux — BIEN DIFFÉRENCIER

### 1. DROITS D'AUTEUR / PUBLISHING (Performing Rights Organizations — PROs)
Qui est payé : les AUTEURS-COMPOSITEURS (songwriters, composers, publishers)
Ce qui est protégé : la COMPOSITION (paroles + mélodie)
Quand ça paie : chaque fois que la chanson est jouée publiquement (radio, TV, streaming, live, commerces)

PROs par pays :
- 🇺🇸 USA : ASCAP, BMI, SESAC
- 🇨🇦 Canada : SOCAN
- 🇫🇷 France : SACEM
- 🇬🇧 UK : PRS for Music
- 🇩🇪 Allemagne : GEMA
- 🇯🇵 Japon : JASRAC
- 🇦🇺 Australie : APRA AMCOS
- 🇧🇷 Brésil : ECAD (UBC, Abramus)
- 🇮🇹 Italie : SIAE
- 🇪🇸 Espagne : SGAE

Identifiant : IPI Number (International Party Information) — numéro unique par auteur
Œuvres identifiées par : ISWC (International Standard Work Code)

### 2. DROITS VOISINS / NEIGHBOURING RIGHTS
Qui est payé : les ARTISTES-INTERPRÈTES (performers, musiciens de session) et les PRODUCTEURS du master
Ce qui est protégé : l'ENREGISTREMENT SONORE (la performance, pas la composition)
Quand ça paie : diffusion radio, TV, lieux publics (PAS le streaming interactif type Spotify)

Sociétés de droits voisins par pays :
- 🇫🇷 France : ADAMI (artistes principaux), SPEDIDAM (musiciens de session), SCPP (producteurs), SPPF (producteurs indépendants)
- 🇬🇧 UK : PPL (performers + labels)
- 🇩🇪 Allemagne : GVL
- 🇯🇵 Japon : CPRA
- 🇧🇷 Brésil : ABRAMUS
- 🇮🇹 Italie : SCF, Nuovo IMAIE
- 🇪🇸 Espagne : AIE (performers), AGEDI (producteurs)
- 🇨🇦 Canada : Re:Sound (collecte), MROC (performers), SOPROQ (producteurs)
- 🇦🇺 Australie : PPCA

Identifiant : ISNI ou IPN (selon la société)
Enregistrements identifiés par : ISRC (International Standard Recording Code)

### 3. DROITS NUMÉRIQUES / DIGITAL PERFORMANCE ROYALTIES
Qui est payé : les ARTISTES-INTERPRÈTES et les PROPRIÉTAIRES DU MASTER (labels/distributeurs)
Ce qui est protégé : l'ENREGISTREMENT SONORE en diffusion numérique
Quand ça paie : radio satellite (SiriusXM), webcasting, radio internet non-interactive

Sociétés de collecte digitale :
- 🇺🇸 USA : SoundExchange (collecte obligatoire pour radio digitale non-interactive)
- 🇬🇧 UK : PPL (couvre aussi le digital)
- 🇫🇷 France : SCPP/SPPF + ADAMI/SPEDIDAM (couvrent aussi le digital)
- International : AARC (Alliance of Artists and Recording Companies)

ATTENTION : SoundExchange ne couvre PAS Spotify/Apple Music (streaming interactif). Ces revenus passent par les distributeurs (DistroKid, TuneCore, CD Baby, etc.)

## Résumé : qui collecte quoi

| Utilisation | Auteur/Compositeur | Artiste/Interprète | Producteur/Label |
|---|---|---|---|
| Radio FM/TV | PRO (SOCAN, ASCAP...) | Droits voisins (PPL, ADAMI...) | Droits voisins (SCPP, PPL...) |
| Streaming Spotify/Apple | PRO + éditeur | Distributeur | Distributeur |
| Radio satellite/webcasting | PRO | SoundExchange (USA) | SoundExchange (USA) |
| Sync (pub/film/série) | Éditeur (publishing) | Label (master) | Label (master) |
| Live/Concert | PRO | — | — |
| Lieux publics (bars, magasins) | PRO | Droits voisins | Droits voisins |

## Rôles et splits typiques
- Songwriter/Composer : 50% publishing (paroles + mélodie)
- Producer : 3-5% master points, parfois publishing share
- Topliner : 10-25% publishing
- Featured artist : 15-25% master royalties
- Mix engineer : flat fee (pas de splits en général)
- Mastering : flat fee

## Sync licensing
- Music supervisor = la personne qui choisit la musique
- Brief = description de ce qu'ils cherchent
- Pitch window = 24-72h en général
- Les premières 8 secondes sont cruciales
- Instrumental > vocal pour les placements TV/pub
- "Stems available" = gros avantage compétitif
- Double clearance nécessaire : publishing (composition) + master (enregistrement)

## Erreurs fréquentes des artistes indépendants
1. Ne pas s'inscrire à une PRO → perdent toutes les royalties de performance
2. Confondre PRO et droits voisins → ne collectent qu'un des trois flux
3. Ne pas enregistrer à SoundExchange (USA) → perdent les royalties radio digitale
4. Pas d'ISRC sur leurs tracks → impossible de tracker les diffusions
5. Splits non documentés → disputes et revenus bloqués en "black box"
