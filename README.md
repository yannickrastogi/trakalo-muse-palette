# trakalog-app — Catalog Manager

Your music catalog, organized & pitch-ready.

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui + Framer Motion
- Supabase (PostgreSQL + Auth + Storage)
- Vercel (deployment)

## Getting Started

Le projet vit dans `~/Desktop/DEV/trakalog-app`.

```sh
git clone <YOUR_GIT_URL> ~/Desktop/DEV/trakalog-app
cd ~/Desktop/DEV/trakalog-app
npm i
npm run dev
```

## Deployment

Deployed via Vercel. Push to `main` to trigger a deploy.
